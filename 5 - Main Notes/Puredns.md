# Installation
## Prerequisites
you need to install massdns through
```
git clone https://github.com/blechschmidt/massdns.git
cd massdns
make
sudo make install
```

## actual installation
```
go install github.com/d3mondev/puredns/v2@latest
--- or 
yay -S puredns # this will install massdns automaticall
```
